Privacy Policy

Effective date: July 17, 2026

1. About this application

AnyaSPB PinFlow (“PinFlow”) is a personal, single-user desktop tool operated by AnyaSPB. It helps the operator prepare selected licensed or self-owned images, create Pins on boards owned by the operator, and save individually selected public Pins through Pinterest's official API.

2. Information processed

PinFlow may process the following information when the operator requests an action:

  • a Pinterest OAuth access token stored locally on the operator's device;
  • board names and identifiers returned temporarily by the Pinterest API;
  • the Pin selected by the operator for creation or saving;
  • a Pixabay or Pexels API key and photo metadata used to find images;
  • locally generated images, previews, and local action timestamps.

PinFlow does not request or collect Pinterest account passwords or browser cookies.

3. How information is used

Information is used only to perform the action explicitly selected by the operator: listing their boards, creating a reviewed Pin, or saving a specific public Pin. PinFlow does not use Pinterest information for advertising, profiling, benchmarking, or activity outside Pinterest.

4. Storage and retention

PinFlow has no remote application server or user database. Credentials are stored only in a local environment file controlled by the operator. Pinterest account content returned by the API is not retained by the application. Locally created images, source attribution, configuration, and action timestamps remain on the operator's device until the operator deletes them.

5. Sharing and sale

PinFlow does not sell, rent, or share personal information or Pinterest API data with third parties. Requests are sent only to Pinterest, Pixabay, or Pexels as needed to provide the selected functionality, subject to their respective privacy policies and terms.

6. Security and access control

The operator is responsible for securing the local device and API credentials. Access to Pinterest is authorized using OAuth permissions and may be revoked through Pinterest account settings. Credentials can also be removed from the local environment file at any time.

7. Children's privacy

PinFlow is not directed to children and is not designed to collect information from children. It is a private tool for the adult operator's own account.

8. Changes to this policy

This policy may be updated if PinFlow's functionality or data practices change. The effective date at the top of this page will be updated when changes are made.
